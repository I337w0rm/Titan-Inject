# 🚀 Titan Inject

A lightweight, modern, and standalone DLL Injector written in pure **C++ (Win32 API)**. 
Titan Inject features a custom Dark Mode GUI, a built-in real-time debugging console, and dual-language support.

## ✨ Features
* **Pure Win32 API GUI:** No bloated external UI frameworks.
* **Custom Dark Mode:** Sleek, modern design with custom-painted controls.
* **Internal Log Console:** Real-time tracking of process selection, errors, and injection status right inside the app.
* **Dual-Language Support:** Easily switch between English and Bosnian on the fly.
* **Live Process List:** Dropdown menu automatically fetches all currently active processes.


---

## 🎮 How to Use
1. Run `TitanInject.exe` **as Administrator** (required for memory injection).
2. Accept the initial disclaimer.
3. Select your target process from the dropdown menu (or click *Refresh* if you just opened the game).
4. Click *Browse...* and locate your target `.dll` file.
5. Click **INJECT**.
6. Check the built-in console at the bottom for success or error logs.
